产生原因是,httpRequest header中 默认是 contentType = application/x-www-form-urlencoded, 如果相传boolean 应该改为 json

```
$.ajax({
		type: 'POST',
		contentType: 'application/json; charset=utf-8',
		url: url,
		data: JSON.stringify(data),
		success: (res) => { }
});
```
